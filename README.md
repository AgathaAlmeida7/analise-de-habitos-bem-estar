# Análise de Hábitos e Bem-Estar

Projeto de análise de dados utilizando dataset público do Kaggle para explorar hábitos de vida e bem-estar.

## Objetivo

Identificar padrões e gerar insights sobre hábitos que influenciam o bem-estar.

## Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Git
- GitHub
